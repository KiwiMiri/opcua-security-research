# 현실 체크 - SignAndEncrypt 설정

## 🔍 문제점

### python-opcua 라이브러리 제한
- ✅ NoSecurity: 작동함
- ⚠️ SignAndEncrypt: 인증서 설정 복잡
- ❌ 실제 프로덕션에서는 CA 인증서 구조 필요

### 현재 상황
1. 서버: SignAndEncrypt 설정 시도 → 실패
2. 클라이언트: SignAndEncrypt 연결 시도 → 실패
3. 결과: NoSecurity만 작동
